Dataset Name
Social Media Usage and Productivity Dataset

Overview
This dataset contains behavioral information about individuals including digital activity patterns, work habits, stress levels, sleep behavior, and productivity scores.

Dataset Size
Approximately 30,000 observations.

Feature Categories

Demographics
Gender, age group, job type

Digital Behavior
Daily social media usage
Notifications per day
Screen time before sleep

Work Patterns
Work hours
Offline hours
Focus app usage

Wellbeing Indicators
Sleep hours
Stress levels
Burnout frequency

Target Variable
Actual productivity score

Potential Limitations

Self-reported behavioral variables may introduce bias.

Data may not represent all geographic or occupational groups.

Some fields contain missing values requiring preprocessing.
